Random Numbers

Random Number (Decimal)

Generates a random decimal number between min (inclusive) and max (exclusive).

function randomBetween(min, max) {
  return Math.random() * (max - min) + min;

Random Integer

Generates a random integer between min and max (both inclusive).

function randomIntBetween(min, max) {
  min = Math.ceil(min);
  max = Math.floor(max);
  return Math.floor(Math.random() * (max - min + 1)) + min;
}

Logarithm with Custom Base

Calculates the logarithm of a number with a specified base.

function logBase(number, base) {
  if (number <= 0) {
    throw new Error("Cannot calculate logarithm of zero or negative number!");
  }
  if (base <= 0 || base === 1) {
    throw new Error("Invalid logarithm base!");
  }
  return Math.log(number) / Math.log(base);
}
