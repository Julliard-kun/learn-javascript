String Rules:

  • Strings are sequences of characters enclosed in single or double quotes.
  • When you used typeof to check a string, you need to use quotes around the string to not get the result "undefined".
  • When you concatenate a string and a number, the number will be converted to a string.
  • Strings are immutable, meaning they cannot be changed after creation.
